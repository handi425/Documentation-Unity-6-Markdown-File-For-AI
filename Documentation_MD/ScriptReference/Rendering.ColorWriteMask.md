### Description

Specifies which color components will get written into the target framebuffer.

### Properties

[Alpha](Rendering.ColorWriteMask.Alpha.html)| Write alpha component.  
---|---  
[Blue](Rendering.ColorWriteMask.Blue.html)| Write blue component.  
[Green](Rendering.ColorWriteMask.Green.html)| Write green component.  
[Red](Rendering.ColorWriteMask.Red.html)| Write red component.  
[All](Rendering.ColorWriteMask.All.html)| Write all components (R, G, B and
Alpha).
