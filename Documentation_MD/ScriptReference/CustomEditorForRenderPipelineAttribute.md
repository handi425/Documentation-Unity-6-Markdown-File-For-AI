**Obsolete** CustomEditorForRenderPipelineAttribute is deprecated. Use
CustomEditor with SupportedOnCurrentPipeline instead. #from(2023.1).

### Description

Tells an Editor class which run-time type it's an editor for when the given
RenderPipeline is activated.

When you make a custom editor for a component that changes based on the active
RenderPipeline you need to put this attribute on the class.  
  
Additional resources: [Editor](Editor.html) class.
