public float height;

### Description

Customizable height of the multi column header.

Default value is
[MultiColumnHeader.DefaultGUI.defaultHeight](IMGUI.Controls.MultiColumnHeader.DefaultGUI-
defaultHeight.html) as this height makes room the for sorting arrow above the
header text. For minimal height use
[MultiColumnHeader.DefaultGUI.minimumHeight](IMGUI.Controls.MultiColumnHeader.DefaultGUI-
minimumHeight.html).  
  
Additional resources: [canSort](IMGUI.Controls.MultiColumnHeader-
canSort.html).
