## Declaration

public bool
SaveToVectorImage([UIElements.VectorImage](UIElements.VectorImage.html)
vectorImage);

### Parameters

vectorImage | The VectorImage object that will be initialized with this painter. This object should not be null.  
---|---  
  
### Returns

**bool** True if the VectorImage initialization succeeded. False otherwise.

### Description

Saves the content of this [Painter2D](UIElements.Painter2D.html) to a
[VectorImage](UIElements.VectorImage.html) object.

The size and content of the vector image will be determined from the bounding-
box of the visible content of the painter object. Any offset of the visible
content will not be saved in the vector image.
