## Declaration

public void CollectElements(HashSet<GraphElement> collectedElementSet,
Func<GraphElement,bool> conditionFunc);

### Parameters

collectedElementSet | The set of matching graph elements.  
---|---  
conditionFunc | A function that determines whether a graph element is added to the set of matching graph elements.  
  
### Description

Retrieves the set of nodes contained in this stack and its edges. The
retrieved graph elements match a specific condition.
