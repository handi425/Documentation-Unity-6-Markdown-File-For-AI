## Declaration

public static [ComputeShader](ComputeShader.html)
CreateComputeShaderAsset([AssetImporters.AssetImportContext](AssetImporters.AssetImportContext.html)
context, string source);

### Parameters

context | The context object the asset system uses to register shader dependencies.  
---|---  
source | The string that contains a shader written in HLSL code.  
  
### Description

Creates a new [ComputeShader](ComputeShader.html) object from the provided
source code string. You can use this method alongside the
[ScriptedImporter](AssetImporters.ScriptedImporter.html) to create custom
compute shader generation tools in the Editor.
