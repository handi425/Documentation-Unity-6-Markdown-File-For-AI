### Description

This interface provides access to a VisualElement transform data.

Reading properties of this object gives information about the transform of its
VisualElement. It is possible to write the position, scale and rotation of the
transform.  
  

### Properties

[matrix](UIElements.ITransform-matrix.html)|  Transformation matrix calculated
from the position, rotation and scale of the transform (Read Only).  
---|---  
[position](UIElements.ITransform-position.html)|  The position of the
VisualElement transform.  
[rotation](UIElements.ITransform-rotation.html)|  The rotation of the
VisualElement transform stored as a Quaternion.  
[scale](UIElements.ITransform-scale.html)|  The scale of the VisualElement
transform.
