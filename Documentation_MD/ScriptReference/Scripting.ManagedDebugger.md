### Description

Representation of managed debugger in UnityEditor.

### Static Properties

[isAttached](Scripting.ManagedDebugger-isAttached.html)| Returns true if there
is a managed debugger attached to the UnityEditor, or false if there is not.  
---|---  
[isEnabled](Scripting.ManagedDebugger-isEnabled.html)| Returns true if managed
debugger is enabled, or false if it is not.  
  
### Static Methods

[Disconnect](Scripting.ManagedDebugger.Disconnect.html)| Disconnects the
managed debugger attached to the UnityEditor.  
---|---  
  
### Events

[debuggerAttached](Scripting.ManagedDebugger-debuggerAttached.html)| This
event triggers whenever managed debugger is attached or detached.  
---|---
