## Declaration

public void SetNeighbors([Terrain](Terrain.html) left, [Terrain](Terrain.html)
top, [Terrain](Terrain.html) right, [Terrain](Terrain.html) bottom);

### Parameters

left | The Terrain tile to the left is in the negative X direction.  
---|---  
top | The Terrain tile to the top is in the positive Z direction.  
right | The Terrain tile to the right is in the positive X direction.  
bottom | The Terrain tile to the bottom is in the negative Z direction.  
  
### Description

Lets you set up the connection between neighboring Terrain tiles. This ensures
LOD matches up on neighboring Terrain tiles.

Note that it isn't enough to call this function on one Terrain; you need to
set the neighbors of each Terrain.
