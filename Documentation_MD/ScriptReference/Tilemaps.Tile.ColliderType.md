### Description

Enum for determining what collider shape is generated for this
[Tile](Tilemaps.Tile.html) by the
[TilemapCollider2D](Tilemaps.TilemapCollider2D.html).

### Properties

[None](Tilemaps.Tile.ColliderType.None.html)| No collider shape is generated
for the Tile by the TilemapCollider2D.  
---|---  
[Sprite](Tilemaps.Tile.ColliderType.Sprite.html)| The Sprite outline is used
as the collider shape for the Tile by the TilemapCollider2D.  
[Grid](Tilemaps.Tile.ColliderType.Grid.html)| The grid layout boundary outline
is used as the collider shape for the Tile by the TilemapCollider2D.
