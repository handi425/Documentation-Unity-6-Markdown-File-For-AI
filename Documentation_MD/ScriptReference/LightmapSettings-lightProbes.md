public static [LightProbes](LightProbes.html) lightProbes;

### Description

Baked Light Probe data.

The data includes: probe positions, Spherical Harmonics (SH) coefficients and
the tetrahedral tessellation. The data is a merged representation of the light
probe data from all loaded scenes. For more information, see
[LightProbes](LightProbes.html).  
  
Use this property to swap between pre-baked [LightProbes](LightProbes.html)
objects at runtime.  
  
Additional resources: [Light Probes in the Unity
Manual](../Manual/LightProbes.html).
