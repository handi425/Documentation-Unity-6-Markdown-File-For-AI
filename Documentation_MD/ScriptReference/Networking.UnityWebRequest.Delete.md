## Declaration

public static [Networking.UnityWebRequest](Networking.UnityWebRequest.html)
Delete(string uri);

### Parameters

uri | The URI to which a `DELETE` request should be sent.  
---|---  
  
### Returns

**UnityWebRequest** A UnityWebRequest configured to send an HTTP `DELETE`
request.

### Description

Creates a UnityWebRequest configured for HTTP `DELETE`.

This method creates a UnityWebRequest, sets the verb to `DELETE` and sets the
target URL to the string argument `uri`. It sets no custom flags or headers.  
  
This method attaches no [DownloadHandler](Networking.DownloadHandler.html) or
[UploadHandler](Networking.UploadHandler.html) to the
[UnityWebRequest](Networking.UnityWebRequest.html).
