## Declaration

protected bool SupportsStageHandling();

### Returns

**bool** True if the Scene view automatically reacts to stage changes.

### Description

Override this method to control whether the Scene view should change when you
switch from one stage to another stage.

By default, the Scene view changes when you switch from one stage to another.
For example, the Scene view changes to render the Prefab Stage when you open
that stage.
