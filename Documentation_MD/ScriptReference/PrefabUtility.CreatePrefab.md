## Declaration

public static [GameObject](GameObject.html) CreatePrefab(string path,
[GameObject](GameObject.html) go,
[ReplacePrefabOptions](ReplacePrefabOptions.html) options);

### Parameters

path | The path where the Prefab is saved.  
---|---  
go | The GameObject that you want to create a Prefab from.  
  
### Returns

**GameObject** A reference to the created Prefab.

### Description

Creates a Prefab from a game object hierarchy.

Use this to create a Prefab from a GameObject in the hierarchy, and give it a
path to save the Prefab in.
