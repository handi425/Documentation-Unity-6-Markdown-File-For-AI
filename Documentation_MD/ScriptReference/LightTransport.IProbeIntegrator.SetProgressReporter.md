## Declaration

public void
SetProgressReporter([LightTransport.BakeProgressState](LightTransport.BakeProgressState.html)
progress);

### Parameters

progress |  [BakeProgressState](LightTransport.BakeProgressState.html) tracker to set on the probe integrator.  
---|---  
  
### Description

Set progress tracker.

Set a [BakeProgressState](LightTransport.BakeProgressState.html) on the
integrator. The integrator will report progress by updating the reporter while
the integration is in progress.

    
    
    var integrator = new [RadeonRaysProbeIntegrator](LightTransport.RadeonRaysProbeIntegrator.html)();		
    var progress = new [BakeProgressState](LightTransport.BakeProgressState.html)();
    integrator.SetProgressReporter(progress);
