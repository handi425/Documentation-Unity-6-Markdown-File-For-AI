### Description

Contains and represents NavMesh data.

An object of this class can be used for creating instances of NavMeshes. See
[NavMesh.AddNavMeshData](AI.NavMesh.AddNavMeshData.html). The contained
NavMesh can be built and updated using the build API. See
[NavMeshBuilder](AI.NavMeshBuilder.html) and methods therein.

### Properties

[position](AI.NavMeshData-position.html)| Gets or sets the world space
position of the NavMesh data.  
---|---  
[rotation](AI.NavMeshData-rotation.html)| Gets or sets the orientation of the
NavMesh data.  
[sourceBounds](AI.NavMeshData-sourceBounds.html)| Returns the bounding volume
of the input geometry used to build this NavMesh (Read Only).
