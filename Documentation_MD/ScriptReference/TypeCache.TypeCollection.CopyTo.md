## Declaration

public void CopyTo(Array array, int arrayIndex);

### Parameters

array | The one-dimensional destination Array to which elements are copied from [TypeCollection](TypeCache.TypeCollection.html).  
---|---  
arrayIndex | The zero-based index where elements are copied to in the destination Array.  
  
### Description

Copies the collection to an array.

The destination array should contain enough space to fit the collection. The
destination array must be of type _Type[]_.
