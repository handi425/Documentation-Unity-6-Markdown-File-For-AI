## Declaration

public static float DefaultAngleHandleSizeFunction([Vector3](Vector3.html)
position);

### Parameters

position | The current position of the handle in the space of [Handles.matrix](Handles-matrix.html).  
---|---  
  
### Returns

**float** The size to use for a handle at the specified position.

### Description

A [SizeFunction](Handles.SizeFunction.html) that returns a fixed screen-space
size.

This method is the default value assigned to
[angleHandleSizeFunction](IMGUI.Controls.ArcHandle-
angleHandleSizeFunction.html) for new
[ArcHandle](IMGUI.Controls.ArcHandle.html) instances, and is not intended to
be called on its own.
