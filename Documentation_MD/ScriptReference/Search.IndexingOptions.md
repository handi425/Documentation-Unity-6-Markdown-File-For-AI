### Description

Use Indexing options to specify the contents of your search index.

### Properties

[None](Search.IndexingOptions.None.html)| Do not include additional indexing
options.  
---|---  
[Types](Search.IndexingOptions.Types.html)| Include additional types of
information in your Asset index.  
[Properties](Search.IndexingOptions.Properties.html)| Include Asset serialized
properties in your index.  
[Extended](Search.IndexingOptions.Extended.html)| Include extended information
in your index such as sub objects in Scene and Prefab Assets.  
[Dependencies](Search.IndexingOptions.Dependencies.html)| Include Asset
dependencies in your index.  
[Keep](Search.IndexingOptions.Keep.html)| Use this option to save the created
index as an Asset file.  
[Temporary](Search.IndexingOptions.Temporary.html)| Save the search index to a
temporary folder that is not managed by Unity.  
[All](Search.IndexingOptions.All.html)| Apply all indexing options.
