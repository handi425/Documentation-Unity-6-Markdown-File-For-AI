public ref [AMD.FSR2CommandExecutionData](AMD.FSR2CommandExecutionData.html)
executeData;

### Description

The mutable state of the current FSR2 object.

To change the results of FSR2, modify this state prior to calling
[GraphicsDevice.ExecuteFSR2](AMD.GraphicsDevice.ExecuteFSR2.html). Additional
resources: [FSR2CommandExecutionData](AMD.FSR2CommandExecutionData.html).
