## Declaration

public static [AI.NavMeshBuildSettings](AI.NavMeshBuildSettings.html)
GetSettingsByID(int agentTypeID);

### Parameters

agentTypeID | The ID to look for.  
---|---  
  
### Returns

**NavMeshBuildSettings** The settings found.

### Description

Returns an existing entry of NavMesh build settings.

If no previously-created settings match the provided agent type ID, the
returned [NavMeshBuildSettings](AI.NavMeshBuildSettings.html) struct will have
the agentTypeID set to -1. See also:
[NavMeshBuildSettings](AI.NavMeshBuildSettings.html).  
  
**Note:** A default entry will always exist for the agentTypeID being 0.
