## Declaration

public bool
GetRenderResolutionFromQualityMode([AMD.FSR2Quality](AMD.FSR2Quality.html)
qualityMode, uint displayWidth, uint displayHeight, out uint renderWidth, out
uint renderHeight);

### Parameters

qualityMode | The input quality mode. See [FSR2Quality](AMD.FSR2Quality.html) for the list of quality modes.  
---|---  
displayWidth | The input display resolution width.  
displayHeight | The input display resolution height.  
renderWidth | The output resolution width calculated.  
renderHeight | The output resolution height calculated.  
  
### Returns

**bool** Returns true on success. False otherwise.

### Description

Queries the resolution configuration from a specified quality mode preset.
