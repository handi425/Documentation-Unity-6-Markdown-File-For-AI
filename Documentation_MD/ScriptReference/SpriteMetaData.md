### Description

Editor data used in producing a Sprite.

Additional resources: [TextureImporter.spritesheet](TextureImporter-
spritesheet.html).

### Properties

[alignment](SpriteMetaData-alignment.html)| Edge-relative alignment of the
sprite graphic.  
---|---  
[border](SpriteMetaData-border.html)| Edge border size for a sprite (in
pixels).  
[name](SpriteMetaData-name.html)| Name of the Sprite.  
[pivot](SpriteMetaData-pivot.html)| The pivot point of the Sprite, relative to
its bounding rectangle.  
[rect](SpriteMetaData-rect.html)| Bounding rectangle of the sprite's graphic
within the atlas image.
