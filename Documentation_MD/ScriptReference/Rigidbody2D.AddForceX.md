## Declaration

public void AddForceX(float force, [ForceMode2D](ForceMode2D.html) mode =
ForceMode2D.Force);

### Parameters

force | The force to add to the X component of the Linear Velocity in the world space of the [Rigidbody2D](Rigidbody2D.html).  
---|---  
mode | The method used to apply the specified force.  
  
### Description

Adds a force to the X component of the
[Rigidbody2D.linearVelocity](Rigidbody2D-linearVelocity.html) only leaving the
Y component of the world space
[Rigidbody2D.linearVelocity](Rigidbody2D-linearVelocity.html) untouched.

Additional resources: [Rigidbody2D.AddForceY](Rigidbody2D.AddForceY.html) and
[Rigidbody2D.AddForce](Rigidbody2D.AddForce.html).
