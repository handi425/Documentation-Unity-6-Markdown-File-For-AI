### Description

Utility class for computing mesh UVs.

This class allows you to compute UVs for procedurally generated meshes.
Computing UVs is also called "unwrapping".  
  
Additional resources: [Mesh](Mesh.html) class, [Generating Lightmap
UVs](../Manual/LightingGiUvs-GeneratingLightmappingUVs.html).

### Static Methods

[GeneratePerTriangleUV](Unwrapping.GeneratePerTriangleUV.html)| Will generate
per-triangle uv (3 UVs for each triangle) with default settings.  
---|---  
[GenerateSecondaryUVSet](Unwrapping.GenerateSecondaryUVSet.html)| Compute a
unique UV layout for a Mesh, and store it in Mesh.uv2.
