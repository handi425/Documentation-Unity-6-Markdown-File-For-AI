## Declaration

public void Update(float deltaTime);

### Parameters

deltaTime | The time delta.  
---|---  
  
### Description

Evaluates the animator based on deltaTime.

Updating the animator with this function might not work well with the physics
engine or any other system that is normally evaluated by the Game loop.
