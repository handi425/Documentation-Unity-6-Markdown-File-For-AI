### Description

Configurations of the [ScrollView](UIElements.ScrollView.html) to influence
the layout of its contents and how scrollbars appear.
[ScrollView.mode](UIElements.ScrollView-mode.html)

The default is
[ScrollViewMode.Vertical](UIElements.ScrollViewMode.Vertical.html).  
  
For more information, refer to [UXML element ScrollView](../Manual/UIE-uxml-
element-ScrollView.html).

### Properties

[Vertical](UIElements.ScrollViewMode.Vertical.html)|  Configure ScrollView for
vertical scrolling.  
---|---  
[Horizontal](UIElements.ScrollViewMode.Horizontal.html)|  Configure ScrollView
for horizontal scrolling.  
[VerticalAndHorizontal](UIElements.ScrollViewMode.VerticalAndHorizontal.html)|
Configure ScrollView for vertical and horizontal scrolling.
