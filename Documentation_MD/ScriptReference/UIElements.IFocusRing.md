### Description

Interface for classes implementing focus rings.

### Public Methods

[GetFocusChangeDirection](UIElements.IFocusRing.GetFocusChangeDirection.html)|
Get the direction of the focus change for the given event. For example, when
the Tab key is pressed, focus should be given to the element to the right.  
---|---  
[GetNextFocusable](UIElements.IFocusRing.GetNextFocusable.html)|  Get the next
element in the given direction.
