### Description

Specifies a draw range of [draw commands](Rendering.BatchFilterSettings.html).

For more information, see [BatchRendererGroup](../Manual/batch-renderer-
group.html).

### Properties

[drawCommandsBegin](Rendering.BatchDrawRange-drawCommandsBegin.html)| The
index of the first draw command inside the BatchCullingOutput.drawCommands
array that this draw range applies to.  
---|---  
[drawCommandsCount](Rendering.BatchDrawRange-drawCommandsCount.html)| The
number of draw commands, starting from BatchDrawRange.drawCommandsBegin, this
draw range applies to.  
[drawCommandsType](Rendering.BatchDrawRange-drawCommandsType.html)| The
BatchDrawCommandType that apply to draw commands in this draw range.  
[filterSettings](Rendering.BatchDrawRange-filterSettings.html)| The
BatchFilterSettings that apply to draw commands in this draw range.
