## Declaration

public bool
TryGetApplyMethodForFieldName([SerializedProperty](SerializedProperty.html)
property, out Action<SerializedProperty> applyMethod);

### Parameters

property | The property that the user is interacting with.  
---|---  
applyMethod | Assign the custom action to be called if the user selects the apply menu command.  
  
### Returns

**bool** True, if the apply menu should be shown for the property.

### Description

Extension hook for an optional property apply menu item.

Called as the context menu is being built to retrieve the method to be called
(if any) when the user selects the 'apply' menu item. The boolean return value
defines if an apply menu item is to be shown.
