### Description

Specifies the composite operation to be used by a
[Collider2D](Collider2D.html).

### Properties

[None](Collider2D.CompositeOperation.None.html)| Indicates that a
CompositeCollider2D will not be used i.e. no composite operation will take
place.  
---|---  
[Merge](Collider2D.CompositeOperation.Merge.html)| Indicates a composite
operation that composes geometry using a Boolean OR operation.  
[Intersect](Collider2D.CompositeOperation.Intersect.html)| Indicates a
composite operation that composes geometry using a Boolean AND operation.  
[Difference](Collider2D.CompositeOperation.Difference.html)| Indicates a
composite operation that composes geometry using a Boolean NOT operation.  
[Flip](Collider2D.CompositeOperation.Flip.html)| Indicates a composite
operation that composes geometry using a Boolean XOR operation.
