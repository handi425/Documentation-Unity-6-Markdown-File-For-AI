## Declaration

public int IndexOf(object item);

### Parameters

item | The object to locate in the collection.  
---|---  
  
### Returns

**int** Returns the zero-based index of the first occurrence of the item.
Returns _-1_ if the item is not found.

### Description

Gets the first occurrence of an item in the
[MethodCollection](TypeCache.MethodCollection.html).
