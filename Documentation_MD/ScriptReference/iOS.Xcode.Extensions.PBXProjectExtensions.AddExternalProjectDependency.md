## Declaration

public static void
AddExternalProjectDependency([iOS.Xcode.PBXProject](iOS.Xcode.PBXProject.html)
proj, string path, string projectPath,
[iOS.Xcode.PBXSourceTree](iOS.Xcode.PBXSourceTree.html) sourceTree);

### Parameters

path | The path to the external Xcode project (the .xcodeproj file).  
---|---  
projectPath | The project path to the new project.  
sourceTree | The source tree the path is relative to. The [PBXSourceTree.Group](iOS.Xcode.PBXSourceTree.Group.html) tree is not supported.  
  
### Description

Adds an external project dependency to the project.
