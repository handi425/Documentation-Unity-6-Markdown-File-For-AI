public [Vector2](Vector2.html) startPosition;

### Description

The start position to slide the [Rigidbody2D](Rigidbody2D.html) from.

This is used when the
[Rigidbody2D.SlideMovement.useStartPosition](Rigidbody2D.SlideMovement-
useStartPosition.html) is true and overrides using the current
[Rigidbody2D.position](Rigidbody2D-position.html).  
  
Additional resources:
[Rigidbody2D.SlideMovement.useStartPosition](Rigidbody2D.SlideMovement-
useStartPosition.html), [Rigidbody2D.Slide](Rigidbody2D.Slide.html) and
[SlideResults](Rigidbody2D.SlideResults.html).
