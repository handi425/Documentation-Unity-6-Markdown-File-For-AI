### Description

Interface for class capable of handling events.

### Public Methods

[HandleEvent](UIElements.IEventHandler.HandleEvent.html)|  Handles an event
according to its propagation phase and current target, by executing the
element's default action or callbacks associated with the event.  
---|---  
[HasBubbleUpHandlers](UIElements.IEventHandler.HasBubbleUpHandlers.html)|
Returns true if event handlers for the event propagation BubbleUp phase, have
been attached to this object.  
[HasTrickleDownHandlers](UIElements.IEventHandler.HasTrickleDownHandlers.html)|
Returns true if event handlers, for the event propagation TrickleDown phase,
are attached to this object.  
[SendEvent](UIElements.IEventHandler.SendEvent.html)|  Sends an event to the
event handler.
