### Description

The list of options available for the 5G Network Slicing app category key. For
more information, refer to [5G Network Slicing App Category (Apple
Developer)](https://developer.apple.com/documentation/bundleresources/entitlements/com_apple_developer_networking_slicing_appcategory)

### Properties

[None](iOS.Xcode.ProjectCapabilityManager.NetworkSlicing5GAppCategoryOptions.None.html)|
Use this flag when additional 5G Network Slicing app category options aren't
necessary.  
---|---  
[Gaming](iOS.Xcode.ProjectCapabilityManager.NetworkSlicing5GAppCategoryOptions.Gaming.html)|
Use this flag to set Gaming option.  
[Communication](iOS.Xcode.ProjectCapabilityManager.NetworkSlicing5GAppCategoryOptions.Communication.html)|
Use this flag to set the Communication option.
