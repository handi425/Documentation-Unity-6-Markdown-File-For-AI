### Description

Enum describing whether a scene should be baked on load.

Additional resources: [Lightmapping.bakeOnSceneLoad](Lightmapping-
bakeOnSceneLoad.html).

### Properties

[Never](Lightmapping.BakeOnSceneLoadMode.Never.html)| Never bake a scene on
load.  
---|---  
[IfMissingLightingData](Lightmapping.BakeOnSceneLoadMode.IfMissingLightingData.html)|
When opening a scene, generate lighting data for the scene, if there is none
already available. This setting can be used when distributing scenes without
lighting data, such that lighting data is automatically generated upon opening
the scene.
