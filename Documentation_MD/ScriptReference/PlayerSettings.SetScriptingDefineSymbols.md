## Declaration

public static void
SetScriptingDefineSymbols([Build.NamedBuildTarget](Build.NamedBuildTarget.html)
buildTarget, string[] defines);

### Parameters

buildTarget | The [NamedBuildTarget](Build.NamedBuildTarget.html).  
---|---  
defines | Symbols for this build target are passed as an array or as a string separated by semicolons.  
  
### Description

Set user-specified symbols for script compilation for the given build target.

To see defines string values go to Edit > Project Settings > Player Settings >
Scripting Compilation > Scripting Define Symbols.  
  
Additional resources: [Platform Dependent Compilation](../Manual/platform-
dependent-compilation.html).
