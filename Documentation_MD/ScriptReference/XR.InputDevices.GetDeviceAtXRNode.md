## Declaration

public static [XR.InputDevice](XR.InputDevice.html)
GetDeviceAtXRNode([XR.XRNode](XR.XRNode.html) node);

### Parameters

node | The XRNode that owns the requested device.  
---|---  
  
### Returns

**InputDevice** An [InputDevice](XR.InputDevice.html) at this [[XR.XRNode].

### Description

Gets the input device at a given [XRNode](XR.XRNode.html) endpoint.

If there is no device at the specified endpoint, the method returns an
InputDevice on which a call to InputDevice.IsValid returns false.
