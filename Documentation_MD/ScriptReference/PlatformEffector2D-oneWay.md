**Obsolete** PlatformEffector2D.oneWay has been deprecated. Use

**Upgrade to[useOneWay](PlatformEffector2D-useOneWay.html)** public bool
oneWay;

### Description

Whether to use one-way collision behaviour or not.

Various behaviour associated with 2D platforms are provided by this effector
such as allowing colliders to pass-through the effector colliders in one-
direction (one-way) as well as controller whether friction or bounce should be
applied.
