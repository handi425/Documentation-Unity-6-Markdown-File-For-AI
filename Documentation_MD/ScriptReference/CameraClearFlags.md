### Description

Values for Camera.clearFlags, determining what to clear when rendering a
[Camera](Camera.html).

Additional resources: [camera component](../Manual/class-Camera.html).

### Properties

[Skybox](CameraClearFlags.Skybox.html)| Clear with the skybox.  
---|---  
[SolidColor](CameraClearFlags.SolidColor.html)| Clear with a background color.  
[Depth](CameraClearFlags.Depth.html)| Clear only the depth buffer.  
[Nothing](CameraClearFlags.Nothing.html)| Don't clear anything.
