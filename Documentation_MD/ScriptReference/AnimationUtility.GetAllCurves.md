## Declaration

public static AnimationClipCurveData[]
GetAllCurves([AnimationClip](AnimationClip.html) clip, bool includeCurveData =
true);

### Description

Retrieves all curves from a specific animation clip.

If `includeCurveData` is false, all retrieved animation curves are returned as
null. Set `includeCurveData` to false to retrieve the list of curves, their
types, and names.
