## Declaration

public delegate Object ResolvePickingWithWorldPositionCallback(int
localPickingIndex, [Vector3](Vector3.html) worldPos, float depth);

### Parameters

localPickingIndex | The 0-based index that specifies what picking index was selected.  
---|---  
worldPos | The world space position where the picking occurred.  
depth | The raw depth value read from the picking render texture.  
  
### Description

The delegate type to return from
[RenderPickingCallback](HandleUtility.RenderPickingCallback.html) through the
[RenderPickingResult.resolverWithWorldPos](RenderPickingResult-
resolverWithWorldPos.html) member, with the additional world space position
and depth information of where the click occurred.

Additional resources: [RenderPickingResult](RenderPickingResult.html).
