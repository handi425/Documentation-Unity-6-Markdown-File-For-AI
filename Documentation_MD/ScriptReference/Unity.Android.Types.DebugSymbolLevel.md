### Description

The type of debug metadata which will be packaged.

Additional resources:
[UserBuildSettings.DebugSymbols.level](Android.UserBuildSettings.DebugSymbols-
level.html)  
  
Additional resources:
[debugSymbolLevel](https://developer.android.com/reference/tools/gradle-
api/7.3/com/android/build/api/dsl/Ndk#debugSymbolLevel\(\))

### Properties

[None](Unity.Android.Types.DebugSymbolLevel.None.html)| No native debug
metadata will be packaged.  
---|---  
[SymbolTable](Unity.Android.Types.DebugSymbolLevel.SymbolTable.html)| Only the
symbol tables will be packaged.  
[Full](Unity.Android.Types.DebugSymbolLevel.Full.html)| The debug info and
symbol tables will be packaged.
