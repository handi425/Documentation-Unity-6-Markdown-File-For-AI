### Description

With the AsyncReadManager, you can perform asynchronous I/O operations through
Unity's virtual file system. You can perform these operations on any thread or
job.

### Static Methods

[CloseCachedFileAsync](Unity.IO.LowLevel.Unsafe.AsyncReadManager.CloseCachedFileAsync.html)|
Closes a file held open internally by the AsyncReadManager.  
---|---  
[GetFileInfo](Unity.IO.LowLevel.Unsafe.AsyncReadManager.GetFileInfo.html)|
Gets information about a file.  
[OpenFileAsync](Unity.IO.LowLevel.Unsafe.AsyncReadManager.OpenFileAsync.html)|
Opens the file asynchronously.  
[Read](Unity.IO.LowLevel.Unsafe.AsyncReadManager.Read.html)| Issues an
asynchronous file read operation. Returns a ReadHandle.  
[ReadDeferred](Unity.IO.LowLevel.Unsafe.AsyncReadManager.ReadDeferred.html)|
Queues a set of read operations for a file once the specified Jobs have
completed.
