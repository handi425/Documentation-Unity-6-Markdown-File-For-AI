### Description

This is the inner representation of the Text input.

### Properties

[allowedCharacters](UIElements.TextValueField_1.TextValueInput-
allowedCharacters.html)|  Method to override to indicate the allowed
characters in the actual field.  
---|---  
[formatString](UIElements.TextValueField_1.TextValueInput-formatString.html)|
Formats the string.  
  
### Public Methods

[ApplyInputDeviceDelta](UIElements.TextValueField_1.TextValueInput.ApplyInputDeviceDelta.html)|
Called when the user is dragging the label to update the value contained in
the field.  
---|---  
[StartDragging](UIElements.TextValueField_1.TextValueInput.StartDragging.html)|
Method called by the application when the label of the field is started to be
dragged to change the value of it.  
[StopDragging](UIElements.TextValueField_1.TextValueInput.StopDragging.html)|
Method called by the application when the label of the field is stopped to be
dragged to change the value of it.  
  
### Protected Methods

[StringToValue](UIElements.TextValueField_1.TextValueInput.StringToValue.html)|
Converts a string to a value type.  
---|---  
[ValueToString](UIElements.TextValueField_1.TextValueInput.ValueToString.html)|
Convert the value to string for visual representation.
