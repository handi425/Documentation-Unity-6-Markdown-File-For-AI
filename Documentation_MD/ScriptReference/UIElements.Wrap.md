### Description

By default, items will all try to fit onto one line. You can change that and
allow the items to wrap as needed with this property.

### Properties

[NoWrap](UIElements.Wrap.NoWrap.html)|  All items will be on one line. Default
Value.  
---|---  
[Wrap](UIElements.Wrap.Wrap.html)|  All items will be on one line. Default
Value.  
[WrapReverse](UIElements.Wrap.WrapReverse.html)|  Items will wrap onto
multiple lines from bottom to top.
