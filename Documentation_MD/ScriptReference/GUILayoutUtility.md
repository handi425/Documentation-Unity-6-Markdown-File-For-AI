### Description

Utility functions for implementing and extending the GUILayout class.

Use this class to make your own GUI layouting code.

### Static Methods

[GetAspectRect](GUILayoutUtility.GetAspectRect.html)| Reserve layout space for
a rectangle with a specific aspect ratio.  
---|---  
[GetLastRect](GUILayoutUtility.GetLastRect.html)| Get the rectangle last used
by GUILayout for a control.  
[GetRect](GUILayoutUtility.GetRect.html)| Reserve layout space for a rectangle
for displaying some contents with a specific style.
