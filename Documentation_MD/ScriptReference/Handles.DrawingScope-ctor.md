## Declaration

public Handles.DrawingScope([Matrix4x4](Matrix4x4.html) matrix);

### Parameters

matrix | The matrix to use for displaying [Handles](Handles.html) inside the scope block.  
---|---  
color | The color to use for displaying [Handles](Handles.html) inside the scope block.  
  
### Description

Create a new DrawingScope and set [Handles.color](Handles-color.html) and/or
[Handles.matrix](Handles-matrix.html) to the specified values.
