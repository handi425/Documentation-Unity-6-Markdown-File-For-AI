### Description

Provides a method to match query text using a fuzzy search algorithm.

This utility class implements the text search functions using approximate
string matching (fuzzy search). Search for a text pattern in a given string.
If there is a match, get the match score. Get the indices in the string where
the pattern was found.  
  
See [FuzzyMatch](Search.FuzzySearch.FuzzyMatch.html) for a complete example.

### Static Methods

[FuzzyMatch](Search.FuzzySearch.FuzzyMatch.html)| Performs a fuzzy search for
a pattern on a string.
