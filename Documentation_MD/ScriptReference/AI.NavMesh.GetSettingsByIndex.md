## Declaration

public static [AI.NavMeshBuildSettings](AI.NavMeshBuildSettings.html)
GetSettingsByIndex(int index);

### Parameters

index | The index to retrieve from.  
---|---  
  
### Returns

**NavMeshBuildSettings** The found settings.

### Description

Returns an existing entry of NavMesh build settings by its ordered index.

If the index is outside the valid range (0, GetSettingsCount-1), the returned
NavMeshBuildSettings struct will have the agentTypeID set to -1.  
  
Additional resources: [NavMeshBuildSettings](AI.NavMeshBuildSettings.html) and
[GetSettingsCount](AI.NavMesh.GetSettingsCount.html).
