### Description

Video codec to use when importing video clips.

### Properties

[Auto](VideoCodec.Auto.html)| Choose the codec that supports hardware decoding
on the target platform.  
---|---  
[H264](VideoCodec.H264.html)| Encode video with the H.264 codec.  
[H265](VideoCodec.H265.html)| Encode video with the H.265 codec.  
[VP8](VideoCodec.VP8.html)| Encode video using the vp8 codec.
