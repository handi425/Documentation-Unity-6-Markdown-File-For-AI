public int targetDisplay;

### Description

For Overlay mode, display index on which the UI canvas will appear.

This setting makes a Canvas render into the specified display. Maximum number
of secondary displays (eg. monitors) supported is 8.  
  
Additional resources: [Display](Display.html) class,
[Camera.targetDisplay](Camera-targetDisplay.html).
