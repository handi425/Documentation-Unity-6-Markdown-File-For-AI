## Declaration

public string GetRequestHeader(string name);

### Parameters

name | Name of the custom request header. Case-insensitive.  
---|---  
  
### Returns

**string** The value of the custom request header. If no custom header with a
matching name has been set, returns an empty string.

### Description

Retrieves the value of a custom request header.

This method retrieves the value of custom (i.e. user-set) request headers.
These are the headers which will be transmitted _to_ the remote server as part
of the HTTP request.  
  
Additional resources:
[SetRequestHeader](Networking.UnityWebRequest.SetRequestHeader.html).
