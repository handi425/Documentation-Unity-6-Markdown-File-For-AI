### Description

Enum that specifies the type of change to the [StickyNote].

### Properties

[Title](Experimental.GraphView.StickyNoteChange.Title.html)| Specifies that
the title of the [StickyNote] has changed.  
---|---  
[Contents](Experimental.GraphView.StickyNoteChange.Contents.html)| Specify
that the content of the [StickyNote] has changed.  
[Theme](Experimental.GraphView.StickyNoteChange.Theme.html)| Specifies that
the visual theme of the [StickyNote] has changed.  
[FontSize](Experimental.GraphView.StickyNoteChange.FontSize.html)| Specify
that the size of the font used by the [StickyNote] has changed.  
[Position](Experimental.GraphView.StickyNoteChange.Position.html)| Specify
that the position of the [StickyNote] has changed.
