public [Android.AndroidColorModeHdr](Android.AndroidColorModeHdr.html)
colorModeHdr;

### Description

Mirrors the Android property `colorMode` based on the `COLOR_MODE_HDR_MASK`
value.

For information about this property, refer to the Android developer
documentation on
[colorMode](https://developer.android.com/reference/android/content/res/Configuration#colorMode)
and
[COLOR_MODE_HDR_MASK](https://developer.android.com/reference/android/content/res/Configuration#COLOR_MODE_HDR_MASK).  
  
Additional resources: [AndroidColorModeHdr](Android.AndroidColorModeHdr.html)
