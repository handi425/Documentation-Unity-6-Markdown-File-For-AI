## Declaration

public stringView Trim(params char[] chrs);

### Parameters

chrs | A set of characters to remove.  
---|---  
  
### Returns

**stringView** A new [StringView](Search.StringView.html).

### Description

Returns a new [StringView](Search.StringView.html) in which all leading and
trailing occurrences of a set of specified characters from the current
[StringView](Search.StringView.html) are removed.
