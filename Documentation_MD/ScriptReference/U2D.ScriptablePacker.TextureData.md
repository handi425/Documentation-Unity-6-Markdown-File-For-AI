### Description

Texture data of a specific sprite useful to get source pixels in textureData
array.

### Properties

[bufferOffset](U2D.ScriptablePacker.TextureData-bufferOffset.html)| Offset of
the TextureData Array from where the actual data for the current texture
starts.  
---|---  
[height](U2D.ScriptablePacker.TextureData-height.html)| Height of the texture.  
[width](U2D.ScriptablePacker.TextureData-width.html)| Width of the texture.
