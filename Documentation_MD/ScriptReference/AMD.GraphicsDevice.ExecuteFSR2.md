## Declaration

public void
ExecuteFSR2([Rendering.CommandBuffer](Rendering.CommandBuffer.html) cmd,
[AMD.FSR2Context](AMD.FSR2Context.html) fsr2Context, ref
[AMD.FSR2TextureTable](AMD.FSR2TextureTable.html) textures);

### Parameters

cmd | The rendering command buffer to record commands into. This call does not execute the command buffer. You must execute the command buffer yourself at any time after this call.  
---|---  
fsr2Context | The source feature context to execute. You must set the parameters for this command in the FSR2Context object prior to this call.  
textures | The texture table, where inputs/outputs are specified for FSR 2.0 to execute.  
  
### Description

Records the execution of FSR 2.0 into a rendering command buffer. This call
does not execute the command buffer, it only appends custom commands into it.
