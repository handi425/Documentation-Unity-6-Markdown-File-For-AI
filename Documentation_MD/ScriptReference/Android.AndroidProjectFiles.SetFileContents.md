## Declaration

public void SetFileContents(string path, byte[] contents);

### Parameters

path | The file to write to. This path is relative to the Gradle project path.  
---|---  
contents | Byte array contents.  
  
### Description

Sets the content of a new file to the specified bytes.

To use this method, you must first declare the new file in
[AndroidProjectFilesModifier.Setup](Android.AndroidProjectFilesModifier.Setup.html)
using the
[AndroidProjectFilesModifierContext.AdditionalOutputs.AddFileWithContents](Android.AndroidProjectFilesModifierContext.AdditionalOutputs.AddFileWithContents.html)
method.
