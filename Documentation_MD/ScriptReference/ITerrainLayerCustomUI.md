### Description

Provides an interface to display a custom [TerrainLayer](TerrainLayer.html) UI
in the Terrain Layers inspector.

Implement this interface on the ShaderGUI class for your custom terrain
shader. This GUI will only be displayed when you select any Terrain Layer in
the Paint Texture tab with the custom shader assigned to the terrain.

### Public Methods

[OnTerrainLayerGUI](ITerrainLayerCustomUI.OnTerrainLayerGUI.html)| Draws the
custom GUI for the terrain layer.
