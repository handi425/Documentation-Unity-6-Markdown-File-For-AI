### Description

The rendering and read/write access mode of a
[GraphicsTexture](Rendering.GraphicsTexture.html).

Additional resources:
[GraphicsTextureDescriptor](Rendering.GraphicsTextureDescriptor.html),
[GraphicsTexture.descriptor](Rendering.GraphicsTexture-descriptor.html).

### Properties

[None](Rendering.GraphicsTextureDescriptorFlags.None.html)| Default mode,
allows this GraphicsTexture to be sampled from.  
---|---  
[RenderTarget](Rendering.GraphicsTextureDescriptorFlags.RenderTarget.html)|
Allows this GraphicsTexture to be set as a render target and be rendered to.  
[RandomWriteTarget](Rendering.GraphicsTextureDescriptorFlags.RandomWriteTarget.html)|
Allows random write access into this GraphicsTexture on Shader Model 5.0 level
shaders.
