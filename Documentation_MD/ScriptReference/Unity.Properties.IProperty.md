### Description

Base interface for working with properties.

This is used to pass or store properties without knowing the underlying
container or value type. " IProperty_1 " Property_2

### Properties

[IsReadOnly](Unity.Properties.IProperty.IsReadOnly.html)|  Gets a value
indicating whether the property is read-only or not.  
---|---  
[Name](Unity.Properties.IProperty.Name.html)|  Gets the name of the property.  
  
### Public Methods

[DeclaredValueType](Unity.Properties.IProperty.DeclaredValueType.html)|
Returns the declared value type of the property.  
---|---  
[GetAttribute](Unity.Properties.IProperty.GetAttribute.html)|  Returns the
first attribute of the given type.  
[GetAttributes](Unity.Properties.IProperty.GetAttributes.html)|  Returns all
attribute of the given type.  
[HasAttribute](Unity.Properties.IProperty.HasAttribute.html)|  Returns true if
the property has any attributes of the given type.
