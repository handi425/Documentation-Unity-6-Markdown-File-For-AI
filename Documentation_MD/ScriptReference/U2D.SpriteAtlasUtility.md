### Description

Utility methods to pack atlases in the Project.

### Static Methods

[CleanupAtlasPacking](U2D.SpriteAtlasUtility.CleanupAtlasPacking.html)|
Cleanup internal states after packing SpriteAtlas assets. Must be called after
a call to SpriteAtlasUtility.PackAllAtlases.  
---|---  
[PackAllAtlases](U2D.SpriteAtlasUtility.PackAllAtlases.html)| Pack all the
SpriteAtlas Assets in the Project for the particular BuildTarget.  
[PackAtlases](U2D.SpriteAtlasUtility.PackAtlases.html)| Pack all the provided
SpriteAtlas for the particular BuildTarget.
