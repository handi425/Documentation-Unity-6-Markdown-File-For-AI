### Description

A script interface for the [text mesh component](../Manual/class-
TextMesh.html).

Additional resources: [text mesh component](../Manual/class-TextMesh.html).

### Properties

[alignment](TextMesh-alignment.html)| How lines of text are aligned (Left,
Right, Center).  
---|---  
[anchor](TextMesh-anchor.html)| Which point of the text shares the position of
the Transform.  
[characterSize](TextMesh-characterSize.html)| The size of each character (This
scales the whole text).  
[color](TextMesh-color.html)| The color used to render the text.  
[font](TextMesh-font.html)| The Font used.  
[fontSize](TextMesh-fontSize.html)| The font size to use (for dynamic fonts).  
[fontStyle](TextMesh-fontStyle.html)| The font style to use (for dynamic
fonts).  
[lineSpacing](TextMesh-lineSpacing.html)| How much space will be in-between
lines of text.  
[offsetZ](TextMesh-offsetZ.html)| How far should the text be offset from the
transform.position.z when drawing.  
[richText](TextMesh-richText.html)| Enable HTML-style tags for Text Formatting
Markup.  
[tabSize](TextMesh-tabSize.html)| How much space will be inserted for a tab
'\t' character. This is a multiplum of the 'spacebar' character offset.  
[text](TextMesh-text.html)| The text that is displayed.
