## Declaration

public static void SetStepLabel(int id, string label);

### Parameters

id | The progress indicator's unique ID.  
---|---  
label | The steps label.  
  
### Description

Sets the label that displays a progress indicator's steps.

This label displays a progress indicator's reported steps. For example, a
progress indicator that reports "Jobs" might display "11/25 Jobs" and a
progress reporting "Assets" might display "8/20 Assets". To report progress in
steps, use [Progress.Report](Progress.Report.html) with the steps variant.  
  
Additional resources: [Progress.GetStepLabel](Progress.GetStepLabel.html),
[Progress.GetCurrentStep](Progress.GetCurrentStep.html),
[Progress.GetTotalSteps](Progress.GetTotalSteps.html),
[Progress.Item.stepLabel](Progress.Item-stepLabel.html),
[Progress.Item.SetStepLabel](Progress.Item.SetStepLabel.html).
