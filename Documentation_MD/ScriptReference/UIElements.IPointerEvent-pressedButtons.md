public int pressedButtons;

### Description

Gets a bitmask that describes the buttons that are currently pressed.

Pressing a mouse button sets a bit. Releasing the button clears the bit. The
left mouse button sets/clears Bit 0. The right mouse button and pen barrel
button set/clear Bit 1. The middle mouse button sets/clears Bit 2. The pen
eraser sets/clears Bit 5. Additional buttons set/clear other bits.
