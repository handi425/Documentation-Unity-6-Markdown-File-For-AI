### Parameters

value | A context object, which is a temporary store that allows you to compare compilation start and finish events.  
---|---  
  
### Description

An event that is invoked on the main thread when the compilation of assemblies
starts.

This is called before the first
[CompilationPipeline.assemblyCompilationStarted](Compilation.CompilationPipeline-
assemblyCompilationStarted.html) event.  
Context object can be used to match
[CompilationPipeline.compilationStarted](Compilation.CompilationPipeline-
compilationStarted.html) and
[CompilationPipeline.compilationFinished](Compilation.CompilationPipeline-
compilationFinished.html) events.
