### Description

Settings to use during the packing process for this
[SpriteAtlas](U2D.SpriteAtlas.html).

### Properties

[blockOffset](U2D.SpriteAtlasPackingSettings-blockOffset.html)| Block offset
to use while packing.  
---|---  
[enableAlphaDilation](U2D.SpriteAtlasPackingSettings-
enableAlphaDilation.html)| Sets the boundary padding pixels alpha to 0 when
packed into a Sprite Atlas.  
[enableRotation](U2D.SpriteAtlasPackingSettings-enableRotation.html)|
Determines if rotating a sprite is possible during packing.  
[enableTightPacking](U2D.SpriteAtlasPackingSettings-enableTightPacking.html)|
Determines if sprites should be packed tightly during packing.  
[padding](U2D.SpriteAtlasPackingSettings-padding.html)| Value to add boundary
(padding) to sprites when packing into the atlas.
