public [UIElements.ICustomStyle](UIElements.ICustomStyle.html) customStyle;

### Description

The custom style properties accessor of a
[VisualElement](UIElements.VisualElement.html) (Read Only).

To get the custom styles properties of an element, call the
[ICustomStyle.TryGetValue](UIElements.ICustomStyle.TryGetValue.html) method to
query the returned object of this property.  
  
For more information about how to use this property, refer to the [Get custom
styles](../Manual/UIE-get-custom-styles.html) manual page.  
  
For a list of all the supported style properties, refer to the [USS properties
reference](../Manual/UIE-USS-Properties-Reference.html) manual page.  
  
Additional resources: [VisualElement.style](UIElements.VisualElement-
style.html), [VisualElement.resolvedStyle](UIElements.VisualElement-
resolvedStyle.html)
