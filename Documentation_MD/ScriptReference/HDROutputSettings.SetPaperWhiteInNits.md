## Declaration

public static void SetPaperWhiteInNits(float paperWhite);

### Parameters

paperWhite | The brightness of paper white in nits.  
---|---  
  
### Description

Sets the base luminance of a white paper surface in nits or candela per square
meter (cd/m2).

The base luminance of a white paper surface in nits or candela per square
meter (cd/m2). Unity uses this value if
[HDROutputSettings.automaticHDRTonemapping](HDROutputSettings-
automaticHDRTonemapping.html) is true. The higher this value, the brighter the
resulting image appears on the HDR display.  
  
This method has been deprecated. Please use
[HDROutputSettings.paperWhiteNits](HDROutputSettings-paperWhiteNits.html)
instead.
