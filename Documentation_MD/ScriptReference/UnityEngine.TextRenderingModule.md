### Description

The TextRendering module provides access to Unity's text rendering system.

### Classes

[Font](Font.html)| Script interface for font assets.  
---|---  
[TextGenerator](TextGenerator.html)| Class that can be used to generate text
for rendering.  
[TextMesh](TextMesh.html)| A script interface for the text mesh component.  
  
### Structs

[CharacterInfo](CharacterInfo.html)| Specification for how to render a
character from the font texture. See Font.characterInfo.  
---|---  
[TextGenerationSettings](TextGenerationSettings.html)| A struct that stores
the settings for TextGeneration.  
[UICharInfo](UICharInfo.html)| Class that specifies some information about a
renderable character.  
[UILineInfo](UILineInfo.html)| Information about a generated line of text.  
[UIVertex](UIVertex.html)| Vertex class used by a Canvas for managing
vertices.  
  
### Enumerations

[FontStyle](FontStyle.html)| Font Style applied to GUI Texts, Text Meshes or
GUIStyles.  
---|---  
[HorizontalWrapMode](HorizontalWrapMode.html)| Wrapping modes for text that
reaches the horizontal boundary.  
[TextAlignment](TextAlignment.html)| How multiline text should be aligned.  
[TextAnchor](TextAnchor.html)| Where the anchor of the text is placed.  
[TextGeneratorType](TextGeneratorType.html)| Defines the types of text
generators to use.  
[VerticalWrapMode](VerticalWrapMode.html)| Wrapping modes for text that
reaches the vertical boundary.
