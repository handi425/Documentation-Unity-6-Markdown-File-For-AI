public [Camera.GateFitMode](Camera.GateFitMode.html) gateFit;

### Description

There are two gates for a camera, the sensor gate and the resolution gate. The
physical camera sensor gate is defined by the [sensorSize](Camera-
sensorSize.html) property, the resolution gate is defined by the render target
area.

To use this property, enable [usePhysicalProperties](Camera-
usePhysicalProperties.html). Additional resources:
[GateFitMode](Camera.GateFitMode.html) enum.
