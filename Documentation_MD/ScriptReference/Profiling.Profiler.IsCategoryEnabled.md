## Declaration

public static bool
IsCategoryEnabled([Unity.Profiling.ProfilerCategory](Unity.Profiling.ProfilerCategory.html)
category);

### Parameters

category | Which category you want to check the state of.  
---|---  
  
### Returns

**bool** Returns whether or not a given
[ProfilerCategory](Unity.Profiling.ProfilerCategory.html) is currently
enabled.

### Description

Returns whether or not a given
[ProfilerCategory](Unity.Profiling.ProfilerCategory.html) is currently
enabled.

Enabled categories emit stats and samples. Each enabled category adds to the
profiler overhead. To disable a category, either close the corresponding chart
in the [ProfilerWindow](ProfilerWindow.html) or call
[Profiler.SetCategoryEnabled](Profiling.Profiler.SetCategoryEnabled.html).
