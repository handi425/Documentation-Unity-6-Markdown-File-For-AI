public [Vector3](Vector3.html) size;

### Description

Describes the dimensions of the shape.

Used only for the primitive shapes: Sphere, Capsule, Box.  
  
• Sphere: size is the dimensions of a box enclosing the sphere (i.e., x, y,
and z are all equal to the diameter).  
  
• Box: size is the dimensions of the box.  
  
• Capsule: size is the dimensions of a box enclosing the capsule (i.e., x and
z are equal to the diameter of the capsule and y is the height).
