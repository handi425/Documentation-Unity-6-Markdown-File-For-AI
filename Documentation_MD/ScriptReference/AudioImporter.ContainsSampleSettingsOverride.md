## Declaration

public bool ContainsSampleSettingsOverride(string platform);

### Parameters

platformGroup | The platform for which to query if this AudioImporter has an override.  
  
See [BuildTargetGroup](BuildTargetGroup.html) for the complete list of
options.  
---|---  
platform | The platform for which to query if this AudioImporter has an override.  
  
See [BuildTargetGroup](BuildTargetGroup.html) for the complete list of options
and type the desired platform enum name in the form of a string.  
  
### Returns

**bool** Returns true if the platform is currently overriden in this
AudioImporter.

### Description

Returns whether a given build target has its sample settings currently
overridden.
