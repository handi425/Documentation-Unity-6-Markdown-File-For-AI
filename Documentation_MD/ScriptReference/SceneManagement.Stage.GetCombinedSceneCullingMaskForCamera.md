## Declaration

public ulong GetCombinedSceneCullingMaskForCamera();

### Returns

**ulong** The combined Scene culling mask for this Stage. Unity uses this mask
on the Scene view Camera for renderer filtering.

### Description

Gets the Scene culling mask from this Stage.

Depending on the currently open Stages, this can either be a Scene culling
mask from a single Scene or a combined Scene culling mask from multiple
Scenes. Unity uses the mask that this function returns on the Scene view
Camera for renderer filtering.
