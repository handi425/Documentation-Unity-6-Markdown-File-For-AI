### Description

Container class that holds the collection of logs generated by an importer
during the import process.

Additional resources:
[AssetImportContext.LogImportError](AssetImporters.AssetImportContext.LogImportError.html),
[AssetImportContext.LogImportWarning](AssetImporters.AssetImportContext.LogImportWarning.html).
