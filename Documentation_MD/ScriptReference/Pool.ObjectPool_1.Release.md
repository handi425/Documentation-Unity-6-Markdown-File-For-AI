## Declaration

public void Release(T element);

### Parameters

element | The instance to return to the pool.  
---|---  
  
### Description

Returns the instance back to the pool. Returning an instance to a pool that is
full will cause the instance to be destroyed.

If the pool has collection checks enabled and the instance is already held by
the pool then an exception will be thrown.
