### Description

Frame Capture mode.

Use the members of the `FrameCaptureMode` enumeration to set the /GPU Frame
Capture Mode/ option in the scheme of an Xcode project generated by Unity.
These options determine whether GPU frames are captured when using the Metal
API.  
  
Additional resources:
[XcScheme.SetFrameCaptureModeOnRun](iOS.Xcode.XcScheme.SetFrameCaptureModeOnRun.html).

### Properties

[Disabled](iOS.Xcode.XcScheme.FrameCaptureMode.Disabled.html)| Disabled.  
---|---  
[Metal](iOS.Xcode.XcScheme.FrameCaptureMode.Metal.html)| Metal.
