## Declaration

public static [AssetBundle](AssetBundle.html)
GetContent([Networking.UnityWebRequest](Networking.UnityWebRequest.html) www);

### Parameters

www | A finished UnityWebRequest object with [DownloadHandlerAssetBundle](Networking.DownloadHandlerAssetBundle.html) attached.  
---|---  
  
### Returns

**AssetBundle** The same as
[DownloadHandlerAssetBundle.assetBundle](Networking.DownloadHandlerAssetBundle-
assetBundle.html)

### Description

Returns the downloaded [AssetBundle](AssetBundle.html), or `null`.

A static function provided for convenience; equivalent to
((DownloadHandlerAssetBundle)www.downloadHandler).assetBundle.
