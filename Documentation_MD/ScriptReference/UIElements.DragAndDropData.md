### Description

Data stored during drag-and-drop operations, enabling information to be
carried throughout the process.

### Properties

[paths](UIElements.DragAndDropData-paths.html)|  A list of paths to the assets
being dragged.  
---|---  
[source](UIElements.DragAndDropData-source.html)|  The object that started the
drag.  
[unityObjectReferences](UIElements.DragAndDropData-
unityObjectReferences.html)|  Unity Object references being dragged.  
[visualMode](UIElements.DragAndDropData-visualMode.html)|  The state of the
current drag operation.  
  
### Public Methods

[GetGenericData](UIElements.DragAndDropData.GetGenericData.html)|  Gets data
associated with the current drag-and-drop operation.  
---|---  
[SetGenericData](UIElements.DragAndDropData.SetGenericData.html)|  Sets data
associated with the current drag-and-drop operation.
