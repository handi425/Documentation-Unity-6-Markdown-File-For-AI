## Declaration

public void
BeginSession([SearchService.ISearchContext](SearchService.ISearchContext.html)
context);

### Parameters

context | The search context.  
---|---  
  
### Description

A function called at the beginning of a search session.

A search session is started when a user inputs the first character in the
search field. This function is not called again until the next search session
is started.
