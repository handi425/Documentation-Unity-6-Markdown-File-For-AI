# Visual Scripting

[com.unity.visualscripting](https://docs.unity3d.com/Packages/com.unity.visualscripting@1.9/manual/index.html)
![](../uploads/Main/iconRel.png)

## Description

Visual scripting is a workflow that uses visual, node-based graphs to design
behaviors rather than write lines of C# script.  
  
Enabling artists, designers and programmers alike, visual scripting can be
used to design final logic, quickly create prototypes, iterate on gameplay and
create custom nodes to help streamline collaboration.  
  
Visual scripting is compatible with third-party APIs, including most packages,
assets and custom libraries.
