# Mathematics

[com.unity.mathematics](https://docs.unity3d.com/Packages/com.unity.mathematics@1.3/manual/index.html)
![](../uploads/Main/iconRel.png)

## Description

Unity’s C# SIMD math library providing vector types and math functions with a
shader like syntax.
