# Debug Metal graphics

Unity’s [Frame Debugger](FrameDebugger.html) supports Metal and should be the
first tool you use to debug graphics issues in a Unity application that uses
Metal. For more in-depth, Metal-specific graphics issues,
[Xcode](https://developer.apple.com/xcode/) provides graphics debugging tools
such as API validation and **shader** A program that runs on the GPU. [More
info](Shaders.html)  
See in [Glossary](Glossary.html#Shader) validation.

**Note** : Enabling validation increases CPU resource intensity, so only
enable it for debugging purposes.
