# Profile rendering

Resources for using the Frame Debugger window and the Frame Timing Manager to
check and debug rendering issues.

---|---

[Frame Timing Manager](frame-timing-manager-landing.html) | Resources for using the Frame Timing Manager window to capture detailed timing data about performance during individual frames in an application.  
[Rendering Profiler module](ProfilerRendering.html) | Reference for the Rendering **Profiler** A window that helps you to optimize your game.
